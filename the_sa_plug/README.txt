THE SA PLUG — FULL PRACTICE PROJECT

This is the fuller multi-page practice build for The SA Plug.

HOW TO VIEW:
1. Unzip the folder.
2. Open index.html in a browser.
3. Click around the navigation.

FILES:
- index.html = homepage
- start-here.html = first-time visitor guide
- downtown.html = downtown / River Walk / Convention Center guide
- ../where-to-eat.html = food recommendation hub (canonical; lives at site root)
- resort-food.html = resort-area food guide
- summer-family.html = family guide
- driver-concierge.html = driver + concierge service page
- business-services.html = Work With Willie / AI services page
- styles.css = shared Fiesta-modern design system
- script.js = demo form behavior

PRACTICE GOALS:
1. Open the folder in Cursor.
2. Edit text in one page.
3. Edit colors/layout in styles.css.
4. Preview locally in browser.
5. Upload to GitHub.
6. Deploy on Netlify.
7. Repeat the same workflow for The AI Plug.

NOTE:
Demo forms do not send yet. In the live deployed site, business-services uses the Business Operator chat plus Netlify backend lead capture instead of third-party form links.
